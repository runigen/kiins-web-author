# meeoocat editor

## install

```bash
npm install
```

## start

```bash
npm start
```

## build

```bash
npm run build   // 운영서버용

npm run build:local   // 로컬서버용

npm run build:dev  // 개빌서버용

npm run build:prd  // 운영서버용
```
